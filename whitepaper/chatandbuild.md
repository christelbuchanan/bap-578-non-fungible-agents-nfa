# ChatAndBuild: Powering the NFA Ecosystem

ChatAndBuild serves as the reference implementation and ecosystem coordinator for the BEP-007 Non-Fungible Agent standard. By providing essential infrastructure, developer tools, and integration services, ChatAndBuild accelerates the adoption and evolution of standardized on-chain agents while pioneering the learning-enabled agent ecosystem.
